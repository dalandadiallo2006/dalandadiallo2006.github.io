-CV Interactif – Maria Dalanda Billy Diallo

 Description
CV personnel interactif développé entièrement en HTML, CSS et JavaScript vanilla,
sans framework. Il présente mon profil, mon parcours académique, mes compétences
techniques, mes loisirs, mes projets et un formulaire de contact, le tout dans
une interface moderne, animée et responsive.

-Fonctionnalités détaillées

 Mode sombre / clair
- Bouton toggle en haut à droite de la navbar
- Changement instantané des couleurs de toute la page
- Icône lune/soleil qui change selon le mode

-Navigation

- Menu fixe en haut de page (navbar)
- Liens qui défilent automatiquement vers chaque section
- Sections : Qualités, Parcours, Loisirs, Projets, Contact

-Compétences
- Barres de progression animées au chargement
- Pourcentages affichés pour chaque compétence
- Compétences : HTML, CSS, JavaScript, Python, C, Programmation, Microsoft Word

-Formulaire de contact
- Champs : Nom complet, Adresse e-mail, Message
- Validation JavaScript des champs obligatoires
- Notification toast en cas de succès (vert) ou d'erreur (rouge)
- Le formulaire se réinitialise après envoi réussi
- Pas d'alertes natives du navigateur

-Design
- Sidebar fixe à gauche avec photo de profil, informations personnelles et profil
- Contenu principal à droite avec toutes les sections
- Animations CSS sur les éléments au survol
- Bouton retour en haut de page
